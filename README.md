# منصة Quartiplay الاستثمارية

منصة استثمار رقمية متكاملة تربط المستثمرين بالشركات الناشئة والمشاريع الواعدة.

## المميزات

### للمستثمرين
- استكشاف فرص استثمارية متنوعة
- الاستثمار بمبالغ مختلفة
- تتبع الاستثمارات والعوائد
- محفظة رقمية آمنة
- سحب الأموال بسهولة

### للشركات
- عرض فرص استثمارية
- جذب مستثمرين
- إدارة الاستثمارات
- تتبع المستثمرين
- دفع العوائد

### للإدارة
- إحصائيات شاملة
- إدارة المستخدمين
- التحقق من الشركات والمستثمرين
- تتبع المعاملات
- سجلات التدقيق

## البنية التقنية

### الخادم (Backend)
- Node.js + Express
- PostgreSQL
- JWT للمصادقة
- Stripe للدفع

### الواجهة الأمامية (Frontend)
- React 18
- React Router
- Axios
- Tailwind CSS

## التثبيت

### المتطلبات
- Node.js 14+
- PostgreSQL 12+
- npm أو yarn

### خطوات التثبيت

1. استنساخ المستودع
```bash
git clone <repository-url>
cd quartiplay
```

2. تثبيت المكتبات
```bash
npm install
cd client && npm install
```

3. إعداد متغيرات البيئة
```bash
cp .env.example .env
# عدّل .env بمعلومات قاعدة البيانات والمفاتيح
```

4. إنشاء قاعدة البيانات
```bash
npm run db:init
```

5. تشغيل الخادم
```bash
npm start
```

6. تشغيل الواجهة الأمامية (في نافذة أخرى)
```bash
cd client
npm start
```

## الاستخدام

### تسجيل حساب جديد
1. اذهب إلى `/register`
2. اختر نوع الحساب (مستثمر أو شركة)
3. أدخل بيانات الحساب

### الاستثمار
1. اذهب إلى صفحة الفرص
2. اختر فرصة استثمارية
3. أدخل مبلغ الاستثمار
4. أكمل عملية الدفع

### إدارة الشركة
1. أنشئ فرصة استثمارية جديدة
2. حدد الهدف والشروط
3. انتظر المستثمرين
4. أدر الاستثمارات

## الأمان

- تشفير كلمات المرور مع bcrypt
- JWT للمصادقة الآمنة
- HTTPS في الإنتاج
- التحقق من صحة البيانات
- حماية من CSRF و XSS
